# 🗳️ Mesa de Partes - Sistema de Voto Electrónico Seguro

![FastAPI](https://img.shields.io/badge/Backend-FastAPI-009688?style=for-the-badge&logo=fastapi&logoColor=white)
![React](https://img.shields.io/badge/Frontend-React_18-61DAFB?style=for-the-badge&logo=react&logoColor=black)
![TypeScript](https://img.shields.io/badge/Language-TypeScript-3178C6?style=for-the-badge&logo=typescript&logoColor=white)
![Arch Linux](https://img.shields.io/badge/Dev_Env-Arch_Linux-1793D1?style=for-the-badge&logo=arch-linux&logoColor=white)

Este repositorio contiene la implementación de una **Mesa de Partes** diseñada para la automatización del voto electrónico. El objetivo principal es mitigar fraudes y reducir tiempos de espera mediante tecnologías de vanguardia.

---

## 🚀 Características Principales

*   **Seguridad Avanzada:** Anonimización de datos mediante **cifrado SHA-256**.
*   **Criptografía Evolutiva:** Generación dinámica de llaves basada en **algoritmos genéticos** para prevenir ataques de fuerza bruta.
*   **Arquitectura Robusta:** Implementación basada en **Clean Architecture** y principios **SOLID**.
*   **Gestión Ágil:** Desarrollo bajo metodología **Scrum** con trazabilidad total en Jira.
